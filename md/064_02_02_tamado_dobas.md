## Támadó dobás

```
TÉ (Támadó Érték) + k20
```

```
Ha legalább 2 támadásod van:
  fix TÉ:-3 minden támadásra
  (az 1. támadásra is)

A levonás NEM additív.
```

```
Példa: TÉ=34

Max 1 támadás
  1.támadás: TÉ:34

Max 3 támadás
  1.támadás: TÉ:31
  2.támadás: TÉ:31
  3.támadás: TÉ:31
```
