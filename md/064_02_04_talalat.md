## Találat ( `TÉ >= VÉ` )

```
Támadó dobás  >=  Védő Érték
```

Találatnak nevezzük azt a támadást, amelynek értéke eléri, vagy meghaladja a célpont **Védő Értékét**. Ilyenkor a támadó [Sebzésdobást](064_02_07_sebzes.md) tesz.

---
### Többszörös találat

```
TÉ >= VÉ különbség

  +5  →  +3 SP
 +10  →  +6 SP
 +15  →  +9 SP
 ...
```

Amennyiben a találat olyan sikeres lett, hogy további `5`-tel nagyobb az ellenfél Védő értékénél, akkor a támadó `+3 SP` bónuszt kap. Ez ismétlődhet, tehát minden további `+5` után jár a `+3 SP`, nincs felső limit.

#### ⚡Példa többszörös találatra

```
TÉ dobás = 60
  vs
VÉ = 49
```

Ekkor `54`-nál és `59`-nél kap bónuszt a támadó, tehát `+6 SP` extra jár sebzésére.

---
### Sebzés

Sikeres találat esetén a támadó [Sebzésdobást](064_02_07_sebzes.md) tesz. A páncél `SFÉ` értéke **mindig érvényesül** – kivéve, ha a támadó sikeresen alkalmazta a [Területre / Pontra támadás](066_05_altalanos_manoverek.md#ter%C3%BCletre--pontra-t%C3%A1mad%C3%A1s) manővert fedetlen terület ellen.

A páncélokról bővebben a [Vértek, páncélok](069_00_vertek_pancelok.md) fejezetben olvashatsz.
