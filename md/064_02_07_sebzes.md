## Sebzés

Amennyiben a korábban leírt módon sikeres [Találatot](064_02_04_talalat.md) értünk el, akkor ezt követően **Sebző Dobást** tehetünk `k20` kockával.

### Páncéldobás

A Sebzésdobással párhuzamosan történik a Találat fejezetben már kitárgyalt [Páncéldobás](064_02_04_talalat.md#p%C3%A1nc%C3%A9ldob%C3%A1s), melyet az áldozat dob.

---
### Sebzésdobás

```
SP = k20
   + Fegyver-sebzés
   + Módosítók
   + Bónuszok
   – (aktuális SFÉ)

Módosítók:
   → Mesterfegyver fortély
   → Erőbónusz
   → Támadás erőből fortély
   → Harci anatómia fortély
     (csak vértmentes ponton)

Bónuszok:
   → Többszörös találat
     +5 túldobás  →  +2 SP
   → Roham
     +5 SP
```

A `k20` dobás eredményéhez hozzáadódik a fegyver sebzés értéke, a fent részletezett módosítók, bónuszok, és egyes fortélyokból adódó extra értékek.

Az végeredményül kapott **Sebzés Pont** (`SP`) értéket a támadó bemondja a támadás [jellegével](064_02_05_fegyver_sebzes_jellege_tipusa.md) együtt (pl. Szúrás).

---
### Támadó dobás bónusza Sebzésdobásra

A **Sebzésdobásra** előnyt adhat az azt megelőző [Támadó dobás](064_02_02_tamado_dobas.md) `k20` kockája - amennyiben a dobás értéke kimondottan magasra sikerült.

```
Támadó dobás k20 kockája

 1-15: Nincs extra hatás
16-19: Előny+1 Sebzésdobásra
   20: Előny+2 Sebzésdobásra
```

---
### Páncél SFÉ levonása SP értékéből

```
SFÉ: páncéltól, támadási típustól
     és a fegyver Átütésétől függ
```

Ha a **Páncéldobás** szerint véd a páncél, akkor a támadás [sebzés jellegével](064_02_05_fegyver_sebzes_jellege_tipusa.md) (pl. szúrás) azonos jellegű páncél `SFÉ` értékét ("szúrás" elleni) kell levonni a sebzés `SP` értékéből.

---
### SP átváltása ÉP sebzésre

Az esetleges páncél `SFÉ` levonása után megnézzük, hogy a maradék `SP` érték milyen `ÉP` sérülést és `VÉ` csökkenést okozott az áldozatnál.

```
SP        ÉP
0      →  0 ÉP
1-5    →  1 ÉP
6-10   →  3 ÉP
11-15  →  5 ÉP
16-20  →  8 ÉP
21-25  → 11 ÉP
26-30  → 15 ÉP
31-35  → 19 ÉP
36-40  → 24 ÉP
41-45  → 29 ÉP
46-50  → 35 ÉP
```

### VÉ csökkenés találatnál

```
VÉ: -3
```

Akkor is, ha volt `ÉP` seb és akkor is ha nem, mert a páncél felfogott mindent `SP`-t.

## Példák sebzésre

### ⚡Példa 1 - páncél nélküli ellenfélre

Tetves gyanútlanul piacozik, mikor nemezise - Rühes - megböki tőrével.\
Az orvgyilkos kivételesen jó Támadást dobott, ez Sebzésdobására is hatással lesz.

```
Tetves
  ÉP: 16 (Edzettség: +2)
```

```
Rühes
 Hosszú kard: +3 +1 (Erő) SP

TÉ dobás (🎲k20): 16❗
 (végső TÉ most nem érdekes)

TÉ dobás (16-19) közt volt
 → Sebzésdobás: k20 Előny+1
```

```
Rühes Sebzése
 11 SP: 🎲k20 (Előny+1): 7, 11
 +4 SP (Kard+Erő)

15 SP  → -5 ÉP, -3 VÉ
```

### ⚡Példa 2 - páncél nélküli ellenfélre

```
Cobaq
  Erő: +5
  Kétkezes kard: +8 SP

Minimum SP: 14  (k20(1) + 13)
 → Minimum ÉP seb: 5
```

Ha Cobaq `7`-nél nagyobbat dob `k20` kockán (`65%`), akkor már `11 ÉP` sebet okozna, ami sok esetben már halálos kategória.

### ⚡Példa 3 - páncélos ellenfélre

Rühes láncingben feszít, mikor Tetves (ősi ellenfele) italozást és beszólogatást követően megböki tőrével.\
Tetves támadása sikeres, jön a Sebzés.

```
Rühes láncing
  SFÉ: Szúró:10, Vágó:7, Zúzó:3
  Lefedettség: 60%
  (torzó: 50%, felkar: 10%)
  
Rühes ÉP: 14  (Edzettség: +1)
```

```
Tetves tőrrel sebez: 🎲k20+1 SP
 → 14 SP  (dobás: 13)
```

```
Rühes 🎲k10 páncéldobása: 40% (4)
 → 40% < 60%
 → védett területet
   talált el Tetves tőre
```

```
Tetves sebzése: Szúró
Rühes Szúró SFÉ: 10
```

```
Végső SP = 14 SP - 10 SFÉ

4 SP →  -1 ÉP, -3 VÉ
```

Látható, hogy a láncing szinte az egész sebzést felfogta, ha ez nincs, akkor Rühesnek `5 ÉP` sebet kellett volna elszenvednie.
