## 🟣 Apró méretű lény (1)

Apró termetű lények - mint például az Erdőpille - számára `1.` Tapasztalati Szinten kötelező fortély. Később nem is vehető fel.

Nyilvánvalóan csak olyan lény veheti fel, amelynek [Faj Hátterénél](../021_faj_hatterek.md) ez szerepel.

### KP bónusz

```
+18 KP bónusz
```

🔆 Speciális fortély, mert nincs `KP` költsége, sőt, felvételéért cserébe **kapsz** `18 KP` bónuszt!

🔆 Csak `1.` tapasztalati szinten vehető fel

🔆 Az így kapott KP [primer ismeretekre](../010_09_primer_szekunder_ismeretek.md) is fordítható.

### ÉP módosító

Az apró méret, alacsonyabb Életerőt is jelent, ezért karaktered `ÉP` értéke alaphelyzetben is jelentősen alacsonyabb.

```
ÉP Alap: -20
ÉP Edzettség szorzó: x2
```

### Példa

```
Edzettség: +3

ÉP = 14 (28-20 + 3x2)
```
