## Beszéd szájról olvasása

Ha nem hallod a beszédet, csak a beszélő szájmozgását látod, úgy is megpróbálkozhatsz a szöveg megértésével.

### Érzékenység Tulajdonságpróba

```
7: Alap Nehézség
   → átlagos táv
   → jó látási viszonyok
   (célpont szájára)
```

### Célszám módosító körülmények

```
-3: Szájról olvasás fortéllyal
+1: nagyobb távolság
+2: nehezen kivehető szájmozgás //
    enyhe köd
+3: még éppen kivehető szájmozgás //
    rossz látási viszonyok
E fölött lehetetlen.
```
