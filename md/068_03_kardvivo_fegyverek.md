## Kardvívó fegyverek

```
MK: Másfélkezes fegyverek
    egy kézzel forgatva

- KÉ:-3,  TÉ:-5,  VÉ:-5
- Átütés megszűnik (ha volt)
- Erőbónusz limit csökken
  (általában 2-re)
- Sebesség marad
```

```
Íves fegyver:
  +2 SP páncélozatlan
  ellenfél ellen
```

```
KF: Különleges felkészítés
  - Tradíció követelmény
    - A: Amund
    - D: Dzsenn
    - S: Slan
    - F: Fejvadász
  - Mf: 2.fok követelmény
```
