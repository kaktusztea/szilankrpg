## Kardvívó fegyverek

```
MK: Másfélkezes fegyverek
    egy kézzel forgatva

- KÉ:-3,  TÉ:-5,  VÉ:-5
- Átütés megszűnik (ha volt)
- Erőbónuszra -3 büntetés,
  azaz a Sebzésre -3 büntetés
- Sebesség, marad
```

```
IV: Íves fegyverek
    Páncélozatlan ellenfél
    ellen +2 SP sebzésbónusz
```

```
KF: Különleges felkészítés
    Tradíció követelmény
    (Slan / Fejvadász)
```
